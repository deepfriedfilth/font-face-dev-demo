Remember when we were limited to a small list of "safe for web use" fonts? Ah, nostalgia…

Basic usage (CSS)

@import url("//hello.myfonts.net/count/2fa7b0");
@font-face {
    font-family: 'AmasisMTStd-Light';
    src: url('fonts/2FA7B0_0_0.eot');
    src: url('fonts/2FA7B0_0_0.eot?#iefix') format('embedded-opentype'),
         url('fonts/2FA7B0_0_0.woff2') format('woff2'),
         url('fonts/2FA7B0_0_0.woff') format('woff'),
         url('fonts/2FA7B0_0_0.ttf') format('truetype');
    font-weight: 400;
    font-style: normal;
}

Basic usage (JS)

<script>
   WebFontConfig = {
      typekit: { id: 'xxxxxx' }
   };

   (function(d) {
      var wf = d.createElement('script'), s = d.scripts[0];
      wf.src = 'https://ajax.googleapis.com/ajax/libs/webfont/1/webfont.js';
      wf.async = true;
      s.parentNode.insertBefore(wf, s);
   })(document);
</script>
